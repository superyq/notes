# Math 常用方法汇总

1. Math.ceil 向上取整

```js
Math.ceil(1.2) // 2
```

2. Math.floor 向下取整

```js
Math.floor(1.2) // 1
```

3. Math.round 四舍五入

```js
Math.round(1.4) // 1
Math.round(1.6) // 2
```

4. Math.random 0-1 随机数

```js
Math.random() // 0.2745798547204079
```

5. Math.max 返回大值

```js
Math.max(1.2, 2) // 2
```

6. Math.min 返回小值

```js
Math.min(1.2, 2) // 1.2
```

7. Math.abs 绝对值

```js
Math.abs(-1) // 1
```

8. Math.sqrt 开平方

```js
Math.sqrt(9) // 3
```

9. Math.pow 求幂

```js
Math.pow(2,3) // 8
```

10. Math.trunc 整数部分

```js
Math.trunc(-1.2) // -1
```
