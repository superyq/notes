# 清除浮动

1. 父级定义伪类(主要解决方式)

```
/*清除浮动代码*/
.clearfloat:after{display:block;clear:both;content:"";visibility:hidden;height:0}
.clearfloat{zoom:1}
```

2. 在结尾处添加空div标签clear:both

3. 父级div定义height

4. 父级div定义overflow:hidden

5. 父级div定义overflow:auto

6. 父级div也一起浮动

7. 父级div定义display:table

8. 结尾处加br标签clear:both
