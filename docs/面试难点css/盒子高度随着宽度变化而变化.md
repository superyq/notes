# 盒子宽度随着高度变化而变化（意思就是宽高比不变）

## 使用一个隐藏的图片来实现

我们在容器内部添加一张符合我们宽高比例的图片，给图片设置宽度100%；高度auto。

```html
<div class='container'>
  <div class='attr'>
    <img src='1.png' alt=''/>
  </div>
</div>
```

```css
.container {
  width: 100%;
}
.attr {
  background: red;
}
.img {
  width: 100%;
  height: auto;
}
```

## padding-bottom

高：宽 比如：高100px，宽50px， width: 50px; padding-bottom: 100px; 

由于padding-bottom的属性值百分比是按照父容器的宽度来计算的，所以这里我们设置容器宽度为父容器的50%；padding-bottom也为50%；就都是父容器宽度的50%；显示出来就是一个完美的正方形了

```html
<div class='container'>
  <div class='attr'></div>
</div>
```

```css
.container {
  width: 80%;
  heigt: 500px;
}
.attr {
  width: 50%;
  height: 0;
  padding-bottom: 50%;
  background: blue;
}
```
