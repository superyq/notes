## 初始化

使用命令行初始化

```BASH
npm init
```

## 登陆

检测是否登陆

```BASH
npm whoami
```

使用以下命令进行`npm`登陆

```BASH
npm login
```

## 发布

使用以下命令进行`npm`发布

```BASH
npm publish
```

### 参数说明

--access public

## 删除发布的包

使用以下命令进行`npm`删除

```BASH
npm unpublish 包名 --force
```
