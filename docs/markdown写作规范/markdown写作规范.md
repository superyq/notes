<h1 align='center'>markdown 写作规范</h1>

## 1. 标题结构格式

* 标题与紧贴的上下正文使用1整行换行隔开

* \#号和文字之间1个空格连接

* 标题层级如下，最多6级

```
# 顶级标题 等价于 title 和 <h1>
## 次级标题 等价于 <h2>
### 3级标题 等价于 <h3>
#### 4级标题 等价于 <h4>
##### 5级标题 等价于 <h5>
###### 6级标题 等价于 <h6>
####### 注：此行错误，标题层级最多6级 没有<h7>标签
```

## 2. 加强和强调规范

* 以下两种写法都是正确的，请统一为第一种，即一律使用 **加强 *强调

```
*emphasize*   **strong** 相当于<strong>标签和<em>标签
_emphasize_   __strong__
```

* 使用~~给文字添加删除线
