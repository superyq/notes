```js
nvm list available //查看可安装NodeJS版本
nvm ls // 已安装node版本
nvm install xx.xx.xx // 安装node版本
nvm uninstall xx.xx.xx // 卸载node版本
nvm use xx.xx.xx // 切换node版本
```
